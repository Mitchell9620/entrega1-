# Colorfly Studio - Generador de Paletas

Aplicación web estática e interactiva desarrollada como MVP para Colorfly Studio, una agencia de branding que necesita generar paletas de colores de forma rápida, visual e intuitiva.

## Objetivo

Permitir al equipo creativo generar paletas de colores aleatorias para acelerar la creación de propuestas visuales iniciales para clientes de branding.

## Funcionalidades principales

- Generación de paletas aleatorias.
- Selección de tamaño de paleta: 6, 8 o 9 colores.
- Visualización de colores en formato HEX y HSL.
- Copia del código HEX al portapapeles.
- Microfeedback visual mediante toast.
- Bloqueo de colores.
- Guardado de paleta en localStorage.
- Render dinámico según el tamaño seleccionado.
- Interfaz responsive para desktop.
- HTML semántico y accesibilidad básica.
